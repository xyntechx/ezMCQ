# ezMCQ

Easily create, save, and do MCQ papers.

## 🎯 Aim

- Learn Solid + Solid Router
- Learn PocketBase

## 🚀 Functionalities

- Create MCQ papers
- Save MCQ papers
- Do MCQ papers
- Check answers / receive results

## 🤖 Technologies
- Solid
- Solid Router
- PocketBase

## 🔨 Usage
```bash
npm i
```

```bash
npm run dev
```

```bash
backend/pocketbase serve
```
